扉 ꜀Fí (fei1)

Radical + residual strokes
⼾ + 8
Unicode code point
U+6249
Fan Wan entry
  • (Vol. 1 Image 35 扉) 戸也
Williams (1856) entry
  • (Page 52 ꜀扉)
    • A wooden door with two leaves door-leaf (Kangxi: 戶扇也, 闔謂之扉, 以木曰扉以葦曰扇);
    • met. a rustic house.

緋 ꜀Fí (fei1)

Radical + residual strokes
⽷ + 8
Unicode code point
U+7DCB
Reading variations
  • fei2 (common in 緋聞)
Fan Wan entry
  • (Vol. 1 Image 35 緋) [絳]色也一曰赤練
Williams (1856) entry
  • (Page 52 ꜀緋)
    • Red or crimson silk (Kangxi: 帛赤色, 絳色又赤練);
    • ꜀fí ꜁pʻò (fei1 pou4 緋袍), carnation robes.
Conway's commentary
  • Fan Wan entry: Corrected 終 to 絳.

裶 ꜀Fí (fei1)

Radical + residual strokes
⾐ + 8
Unicode code point
U+88F6
Fan Wan entry
  • (Vol. 1 Image 35 [裶]) 曳衣也又衣長貌
Williams (1856) entry
  • (Page 52 ꜀裶)
    • To lift up the skirt;
    • long, trailing, as dresses (Kangxi: 衣長貌, 曳衣貌);
    • ꜀fí ꜀í (fei1 ji1 裶衣), to hold up the a dragging dress.
Conway's commentary
  • Fan Wan entry: Fixed missing stroke in headword 裶.

非 ꜀Fí (fei1)

Radical + residual strokes
⾮ + 0
Unicode code point
U+975E
Fan Wan entry
  • (Vol. 1 Image 35 非) 不是也
Williams (1856) entry
  • (Page 52 ꜀非)
    • Not right, not good (Kangxi: 不是也);
    • not so, opposite (Kangxi: 違也);
    • false, bad, wrong;
    • shameless, low (Kangxi: 下也);
    • to slander (Kangxi: 訾也), to reproach (Kangxi: 責也);
    • ꜁'m ꜀chí shí꜅ ꜀fí (m4 zi1 si6 fei1 唔知是非), I don't know whether it is so or not;
    • ꜂kong shí꜅ ꜀fí (gong2 si6 fei1 講是非), to asperse, tittle-tattle;
    • ꜀fí ꜀heng ꜀yung ꜁yung í꜅ (fei1 heng1 jung4 ji6 非輕容易) this is not so easy;
    • ꜀fí ꜃lai (fei1 lai5 非禮), improper, immodest;
    • ꜂hí ꜀fí (hei2 fei1 豈非), is it not so! i.e. it is true;
    • ꜀fí ꜀shéung ꜁shéung (fei1 soeng4 非常), uncommon, superior;
    • ꜀fí ꜃yá (fei1 jaa5 非也), not so.
Conway's commentary
  • Williams' 非輕容易: Corrected ꜀yung (jung1) to ꜁yung (jung4).
  • Williams' 非禮: Now to molest.
  • Williams' 非常: Corrected ꜀shéung (soeng1) to ꜁shéung (soeng4).
Conway's supplementary entry
