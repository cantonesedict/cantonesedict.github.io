Radical + residual strokes

Unicode code point
U+5118
Alternative forms

Reading variations

Fan Wan entry
  • (Vol. 2 Image 53 儘) 皆也
Williams (1856) entry
  • (Page 601 ꜂儘)
    • Same as tsun꜅ (zeon6) 盡 to finish;
    • empty;
    • all (Kangxi 盡: 皆也);
    • the utmost degree (Kangxi 盡: 極也, 盡之也);

  • Williams' "Same as […]": This is a misreading of Kangxi 同盡, which is only intended for the reading ꜂tsun (zeon2).
Conway's supplementary entry
  1. To allow, to let (Kangxi 盡: 任也, 縱令也, 俗作儘).
