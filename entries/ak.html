Ak꜆. (ak1 高陰入)

厄 ak꜆ (ak1)

Unicode code point
(U+5384)
Alternative forms
(戹)
Reading variations
(aak1: longer syllable)
Williams (1856) entry
(Page 3)
  • Knots in a tree;
  • small door in a house;
  • impeded, cramped;
  • to be ill-used, to be straitened;
  • ak꜆ ꜁kʻung (ak1 kung4 厄窮), poor;
  • ꜁ngai ak꜆ (ngai4 ak1 危厄) in danger.
Conway's commentary
  • Most modern speakers prefer the longer syllable ák꜆ (aak1); see 厄 aak1.
  • Williams erroneously has 𢨪 = ⿸戶巳 for an alternate form instead of 戹.

幄 ak꜆ (ak1)

Unicode code point
(U+5E44)
Williams (1856) entry
(Page 3)
  • A curtain, a screen;
  • a large tent, a markee;
  • to protect, to shelter.

握 ak꜆ (ak1)

Unicode code point
(U+63E1)
Reading variations
(aak1: longer syllable)
Williams (1856) entry
(Page 3)
  • To grasp, to hold within the hand;
  • a little, as much as the palm can hold;
  • ak꜆ teng꜅ (ak1 deng6 握定) to grasp firmly;
  • ak꜆ ꜂shau (ak1 sau2 握手), to shake hands;
  • ak꜆ ꜁chʻí (ak1 ci4 握持), to hold in the hand;
  • ak꜆ iú꜄ (ak1 yiu3 握要) necessary.
Conway's commentary
  • Most modern speakers prefer the longer syllable ák꜆ (aak1); see 握 aak1.

渥 ak꜆ (ak1)

Unicode code point
(U+6E25)
Williams (1856) entry
(Page 3)
  • To moisten, to soak;
  • to enrich with favors;
  • to fertilize;
  • to cover or daub thick;
  • shining, rich;
  • ak꜆ ꜁mung (ak1 mung4 渥蒙), deeply grateful for;
  • ꜀yan ak꜆ (yan1 ak1 恩渥) imperial favor;
  • ak꜆ ꜀tán (ak1 daan1 渥丹), to make very red.

阨 ak꜆ (ak1)

Unicode code point
(U+9628)
Reading variations
(aak1: longer syllable)
Williams (1856) entry
(Page 3)
  • An obstruction, a stop;
  • a defile, a pass;
  • a dangerous path;
  • embarrassed, distressed, poor;
  • kwʻan꜄ ak꜆ (kwan3 ak1 困阨) hampered, restrained.
Conway's commentary
  • Most modern speakers prefer the longer syllable ák꜆ (aak1); see 阨 aak1, for the colloquial "deceive" sense.
