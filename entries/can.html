嗔 ꜀Chʻan (can1)

Radical + residual strokes
口 + 10
Unicode code point
U+55D4
Fan Wan entry
    (Vol. 1 Page 81 嗔) 怒也
Williams (1856) entry
  • (Page 17 ꜀嗔)
    • To get angry;
    • passionate, angry;
    • to rail;
    • ꜀shang ꜀chʻan (sang1 can1 生嗔), to get into a rage;
    • ꜀chʻan kwái꜄ (can gwaai3 嗔怪) to scold.

    Also read ꜁tʻín (tin4); (see 嗔 tin4).

Conway's commentary
  • Fan Wan and Williams both print this as ⿰口眞, but no separate code point to 嗔 U+55D4 exists for this form (blame Han unification inconsistency).

瞋 ꜀Chʻan (can1)

Radical + residual strokes
目 + 10
Unicode code point
U+778B
Fan Wan entry
    (Vol. 1 Page 81 瞋) 怒而張目也
Williams (1856) entry
  • (Page 18 ꜀瞋)
    • To stare at angrily, to glare at;
    • ꜀chʻan nò꜅ ꜁mò tò꜅ (can1 nou6 mou4 dou6 瞋怒無度) angry beyond measure.
Conway's commentary
  • Fan Wan and Williams both print this as ⿰目眞, but no separate code point to 瞋 U+778B exists for this form (blame Han unification inconsistency).
