꜀Chʻan ꜀Tsʻan (can1 陰平)

  • 嗔
  • 瞋

嗔 ꜀Chʻan (can1)

Radical + residual strokes
口 + 10
Unicode code point
U+55D4
Fan Wan entry
  • (Vol. 1 Page 81 嗔) 怒也
Williams (1856) entry
  • (Page 17 ꜀嗔)
    • To get angry;
    • passionate, angry;
    • to rail;
    • ꜀shang ꜀chʻan (sang1 can1 生嗔), to get into a rage;
    • ꜀chʻan kwái꜄ (can gwaai3 嗔怪) to scold.

    Also read ꜁tʻín (tin4); (see 嗔 tin4).

Conway's commentary
  • Fan Wan and Williams both print this as ⿰口眞, but no separate code point to 嗔 U+55D4 exists for this form (blame Han unification inconsistency).

瞋 ꜀Chʻan (can1)

Radical + residual strokes
目 + 10
Unicode code point
U+778B
Fan Wan entry
  • (Vol. 1 Page 81 瞋) 怒而張目也
Williams (1856) entry
  • (Page 18 ꜀瞋)
    • To stare at angrily, to glare at;
    • ꜀chʻan nò꜅ ꜁mò tò꜅ (can1 nou6 mou4 dou6 瞋怒無度) angry beyond measure.
Conway's commentary
  • Fan Wan and Williams both print this as ⿰目眞, but no separate code point to 瞋 U+778B exists for this form (blame Han unification inconsistency).

꜁Chʻan (can4 陽平)

  • 蔯
  • 陳

蔯 ꜁Chʻan (can4)

Radical + residual strokes
艸 + 11
Unicode code point
U+852F
Fan Wan entry
(Not present)
Williams (1856) entry
  • (Page 18 ꜁蔯)
    • A medicinal herb, good for rheumatism;
    • ꜀yan ꜁chʻan (jan1 can4 茵蔯), a plant like origanum.

陳 ꜁Chʻan (can4)

Radical + residual strokes
阜 + 8
Unicode code point
U+9673
Fan Wan entry
  • (Vol. 1 Page 82 陳) 列也張也敷也告也又姓
Williams (1856) entry
  • (Page 18 ꜁陳)
    • To arrange, to set in order, to spread out;
    • to say, to reply;
    • to state, to express;
    • to memorialize;
    • a long time, an old thing;
    • many, all;
    • a feudal state near the present capital of Honán (河南);
    • tsoi꜅ ꜁chʻan [tsʻüt]꜇ ꜁léung (zoi6 can4 zyut6 loeng4 在陳絕糧), to lack food in Chʻin—to be in straits;
    • ꜀pʻò ꜁chʻan (pou1 can4 鋪陳), to seat or rank in order;
    • ꜁chʻan chʻít꜆ (can4 cit3 陳設) to arrange in order;
    • ꜁shéung ꜂piú ꜁chʻan ꜁tsʻing (soeng5 biu2 can4 cing4 上表陳情), to state one's thoughts to H. I. M.;
    • ꜁chʻan ꜁pʻí kòm꜄ kau꜅ (can4 pei4 gam3 gau6 陳皮咁舊) old as a dried orange skin, faded.
Conway's commentary
  • Williams' 在陳絕糧: Corrected tsʻüt꜇ (cyut6) to tsüt꜇ (zyut6).
